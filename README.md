# dProxy
A modified version of RoProxy-Lite made for self-hosting. It's also not for Roblox. It's for Discord.

Setup is easy, simply deploy with the button below and configure environment variables "TIMEOUT" (in seconds) and "RETRIES" (attempts for reconnecting).

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)
